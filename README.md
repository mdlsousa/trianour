# TriaNox — Premium Nutrition Experience Mockup

High-fidelity, fully navigable frontend prototype for investor and UX validation.

## Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide Icons
- shadcn-style UI components

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Navigation map

| Route | Experience |
|-------|------------|
| `/` | Landing page (investor pitch) |
| `/nutritionist/dashboard` | Nutritionist portal |
| `/patient` | Patient mobile-first app (Ricardo Silva) |
| `/patient/meals` | Visual meal plan |
| `/patient/ai` | Conversational AI support |
| `/patient/family` | Family nutrition |
| `/patient/premium` | Premium upsell |
| `/patient/recipes` | Recipe library |
| `/patient/shopping` | Interactive shopping list |
